## 【第1幕：問いかけ】(2-3分)

### 1. つかみ (30秒)

2つのボールを同じ高さからボールを落とすと、必ず同じ速度で地面に着きますよね。
ガリレイの時代に実証されたような、ものすごく当たり前に思える現象です。
これは位置エネルギーと運動エネルギーを足した値が必ず一定になる、
というエネルギー保存の法則から導くことができます。

でも、なぜエネルギーは保存するのでしょうか？
法則というくらいだから、そうなることが物理の「当たり前」というような気もします。
実はこの「当たり前」の裏に物理学で最も重要な考えのひとつである、「対称性」が潜んでいます。
対称性と保存則を結びつけるネーターの定理。それを一緒に探っていきましょう。

## 【第2幕：探求と発見】(7-9分)

### 4. 自由落下の実験 (2分)

まずは実験から始めましょう。
ボールを地面から高さhの位置に置きます。

**[スローモーション再生。エネルギーバーを表示]**

落ちていく間、2種類のエネルギーが変化しています。
まず位置エネルギー。高いところにあるほど大きな位置エネルギーを持っています。式で書くと、mghです。

そして運動エネルギー。速く動いているほど大きなエネルギーです。
式で書くと、½mv²です。

落下すると位置エネルギーが減っていって運動エネルギーが増えていきます。
でもこの2つの合計はずっと一定なんです！

**[画面に式を表示]**

```
E = ½mv² + mgh = 一定
```

位置エネルギーと運動エネルギーの合計は運動の間変わらない。これがエネルギー保存則です。

地面に着く時、位置エネルギーがゼロになって全部が運動エネルギーになります。
このとき速度は√(2gh)と計算できます。


でもなぜエネルギーは保存されるんでしょうか？
ここからが今日の本題です。

### 5. 時間対称性の発見 (2-3分)

**[複数の時計を空間に出現させる]**

ここで素朴だけど重要な事実について考えてみましょう。
今日の0時にこのボールを落としてみます。
地面に着く速度は、√(2gh)です。
では同じ日の12時に落としてみましょう。
結果はもちろん同じで地面に着く時の速度は√(2gh)です！

では明日の朝6時なら？
言うまでもなく結果は同じです！

何時に実験しても結果は変わりません。
時間が変わっても結果は影響を受けない。これが時間対称性です。
「いつやっても同じ」ということを少し専門的な言葉を使うとすると「時間に対して対称的」ということができるのです。

**[画面に図解]**
```
時刻t   → 落下 → 速度 v
時刻t+1 → 落下 → 速度 v  ← 同じ！
時刻t+2 → 落下 → 速度 v  ← 同じ！
```

「そんなこと当たり前じゃん！」と思うかもしれません。
でもネーターの定理はこの時間対称性がエネルギー保存則を生み出していると教えてくれるんです。

---

### 6. 空間対称性の破れ (2分)

**[ボールを異なる高さに配置]**

時間を変えても結果は同じでした。
では高さを変えるとどうなるでしょう？

高さhから落とすと速度は√(2gh)でした。
少しだけ高い場所から落とすと地面に着くときの速度も少しだけ増えます。

あれ？高さが違うと結果が変わりますね？
つまり高さ方向には対称性がないんです。

**[地球の図を表示。重力の矢印]**

なぜか？それは地球の重力があるからです。
地球の重力は高さによって位置エネルギーを変化させます。
だから空間は鉛直方向に一様ではない。

**[対比図を表示]**

でも、ちょっと待ってください。
ボールを横方向にずらしてみましょう。

**[横にずらして落とす]**

重力は鉛直下向きに一様にかかるの横にずらしても落下の結果は変わりません。
つまり横方向には対称性があるんです。

**[まとめの図]**
```
✅ 時間をずらす  → 結果同じ → 対称性あり
✅ 横にずらす    → 結果同じ → 対称性あり
❌ 高さを変える  → 結果変わる → 対称性なし
```

**[カメラに向かって]**

重力が鉛直方向の空間対称性を破っているんです。
そして対称性がない方向には対応する保存則もありません。
逆に言えば対称性があるところに保存則が生まれるんです。
これがネーターの定理の核心です！

#### 7-5. 保存量の導出 (1分30秒)

**[ネーターの肖像を再び表示]**

さて、ネーターはこう考えました。

**[吹き出し表示]**

「ラグランジアンが時間で変わらないなら...何か変わらない量があるはずだ！」

**[画面に公式を表示]**

そして、彼女が発見した公式がこれです：

```
E = v · (∂L/∂v) - L
```

**[カメラに向かって]**

この式で計算される量**E**が、保存されるんです。

そして、この**E**こそが、**エネルギー**なんです！

**[黒板風ボードを出現]**

では、実際に計算してみましょう。

式の変形はちょっと難しいので、ゆっくり見ていきますね。

**[Step 1を表示]**

**ステップ1**: まず、∂L/∂vを計算します。

```
L = ½mv² - mgh

∂L/∂v = ∂/∂v[½mv² - mgh]
```

**[vで微分する部分を黄色でハイライト]**

vで微分するので、½mv²の部分だけが残ります。

```
∂L/∂v = mv
```

**[指差しながら]**

速度で微分すると、**運動量mv**が出てきました！

**[Step 2を表示]**

**ステップ2**: これを使って、Eを計算します。

```
E = v · (∂L/∂v) - L
  = v · (mv) - [½mv² - mgh]
```

**[計算過程をアニメーション]**

```
  = mv² - ½mv² + mgh
```

**[mv²とー½mv²を強調]**

mv²から½mv²を引くと、½mv²が残ります。

```
  = ½mv² + mgh
```

**[大きく表示、紫色で強調]**

```
E = ½mv² + mgh
```

**[カメラに向かって、笑顔]**

これが、エネルギーの式です！

運動エネルギー + 位置エネルギー

**[矢印を表示]**

```
時間対称性 (∂L/∂t = 0)
    ↓
エネルギーE が保存される
```

**[カメラに向かって、強調]**

時間対称性から、エネルギー保存則が**必然的に**導かれました。

これが、ネーターの定理です！
